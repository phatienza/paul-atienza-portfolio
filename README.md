# Paul Henry Atienza — Portfolio

Personal portfolio website built with Next.js and Tailwind CSS.

## 🚀 Live Website
https://www.paulatienza.dev/

## 🛠 Tech Stack
- Next.js
- React
- Tailwind CSS
- TypeScript
- Vercel (Deployment)

## 📌 Features
- Responsive design
- Sticky navigation
- Smooth scroll animations
- Professional experience & certifications section
- CI/CD via GitHub + Vercel

## 📦 Deployment
Automatically deployed through Vercel on every push to `main`.
